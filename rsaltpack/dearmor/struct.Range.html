Unstable (step_by)

: recent addition

Creates an iterator with the same range, but stepping by the given amount at each iteration.

The resulting iterator handles overflow by stopping.

Examples

#![feature(step_by)]

for i in (0..10).step_by(2) {
    println!("{}", i);
}
